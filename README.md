# Poked-x---Little-Project_Manual-do-Dev
Projeto de Pokédex feito com HTML, CSS e JavaScript, seguindo o tutorial do Manual do Dev. A aplicação lista pokémons, exibe detalhes usando a PokéAPI e utiliza manipulação de DOM para atualizar as informações de forma dinâmica e interativa.

Este projeto é uma Pokédex desenvolvida em HTML, CSS e JavaScript, baseada no processo mostrado no vídeo do canal Manual do Dev. A proposta foi acompanhar o desenvolvimento apresentado e reproduzir a aplicação aplicando entendimento próprio de cada etapa. O resultado é uma interface leve, responsiva e totalmente funcional, capaz de exibir diferentes pokémons e suas principais informações.

A estrutura do projeto foi construída com HTML para organizar os elementos que compõem a Pokédex, como a área de listagem dos pokémons, a seção para a imagem principal e os textos informativos. O CSS foi responsável por definir o layout, as cores, o estilo dos cards e a adaptação visual para diferentes tamanhos de tela, deixando a experiência mais agradável e intuitiva.

A lógica da aplicação foi feita em JavaScript, manipulando o DOM para atualizar dinamicamente os dados exibidos. O código realiza requisições à API de Pokémons (PokéAPI), obtendo informações como nome, número, tipo, imagem, altura, peso e demais atributos. Cada pokémon pode ser visualizado individualmente, permitindo ao usuário navegar pela Pokédex de maneira simples e direta.

Seguindo o fluxo do tutorial, o projeto também explora conceitos essenciais como criação de funções, tratamento de respostas de API, estruturação de dados e interação via eventos de clique ou navegação. Além disso, a construção passo a passo ajudou a reforçar práticas importantes para projetos front-end: organização de arquivos, modularidade, clareza no CSS e uso eficiente de JavaScript.

Este projeto foi criado como prática de estudo e fortalecimento de lógica, manipulação de DOM e integração com APIs. Ele serve tanto como aprendizado pessoal quanto como base para evoluções futuras, como filtros por tipo, modo escuro, animações, paginação, busca por nome/ID ou uma interface inspirada no design dos jogos clássicos.
